### 1. 初始專案需求
**👤 User:** 
建立一個完整的 Python 專案，用於透過 CWA API（特別是 F-A0010-001 農業氣象預報端點）抓取台灣北部、中部、南部、東北部、東部與東南部地區的 7 天天氣預報資料，解析 JSON 回應以提取溫度資料（最低與最高溫），並將其儲存為名為「weather_data.csv」的 CSV 檔案。同時，建立一個互動式的 Streamlit 網頁應用程式，採用左右分欄布局：左側顯示基於 Folium 的台灣地圖，並依據平均溫度以不同顏色的圓形標記各地區（藍色 <20°C、綠色 20–25°C、黃色 25–30°C、紅色 >30°C）；右側提供日期選擇下拉選單，可動態篩選資料，並顯示該日期的溫度資料表。確保應用程式能處理 SSL 驗證問題，為各區域使用大致座標作為標記位置，並在地圖標記中加入顯示區域詳細資訊的彈出視窗；同時建立 Python 虛擬環境、安裝必要套件...
